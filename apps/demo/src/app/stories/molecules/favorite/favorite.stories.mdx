## Molecule favorite overview

> The favorite molecule is a component that allows the user to add or remove a favorite.

<br />
<br />

### Composition

The favorite molecule is composed of a button and a label:

- The button is an icon button that allows the user to add or remove a favorite.
- The label is a counter of users liking the content.

<br />

### Inactive

> The favorite molecule is inactive when the user has not added a favorite.
